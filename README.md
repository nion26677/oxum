# 🦀 Oxum

Легковесный, расширяемый тайлинговый оконный менеджер для X11, написанный на **Rust** с использованием библиотеки `x11rb`.

## 🎯 Философия проекта

Проект вдохновлен легендарным оконным менеджером `dwm`, но решает его главную боль — сложность кастомизации. Вместо ручного наложения `.diff` файлов и разрешения бесконечных merge-конфликтов, этот ВМ спроектирован для **легкого патчинга и модификации** благодаря модульной архитектуре Rust и строгой типизации.

* **Безопасность и скорость:** Благодаря Rust, никаких сегфолтов (Segmentation faults) при управлении окнами.
* **Понятный код:** Чистая кодовая база на основе `x11rb` вместо старой и небезопасной `Xlib`.
* **Простой патчинг:** Модифицируйте поведение, добавляя модули или меняя конфигурацию прямо в исходном коде, не боясь сломать базовый функционал.
* **Конфигрурация:** Используйте для настройки статический или динамический конфиг и настраивайте поведвения прямо на лету.

## 🛠 Стек технологий

* **Язык:** [Rust](https://www.rust-lang.org/) (Safe & Fast)
* **Протокол:** X11 через библиотеку [x11rb](https://github.com/psychon/x11rb)
* **Сборщик задач:** [just](https://github.com/casey/just)

## 📦 Установка

### Требования
Перед установкой убедитесь, что в вашей системе установлены:
* Компилятор Rust (`cargo`, `rustc`)
* Утилита автоматизации `just`
* Заголовочные файлы X11 (обычно идут вместе с X-сервером)

### Сборка и установка

Проект использует `just` для автоматизации рутинных задач. Для компиляции и установки бинарного файла в систему выполните:

```bash
just install
```

Примечание: Команда скомпилирует проект в режиме --release и скопирует готовый бинарный файл в директорию /usr/bin/ (может потребоваться пароль sudo).

# 🚀 Запуск
Добавьте следующую строку в ваш файл `~/.xinitrc`, чтобы запускать менеджер окон при старте X-сессии:
```bash
exec oxum
```
